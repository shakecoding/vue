▶ SPA (Single Page Application)
    단일 페이지로 구성된 웹 애플리케이션
    SPA는 처음에 웹 애플리케이션의 모든 콘텐츠를 한 번에 로드하고,
    이후 페이지 간의 이동이나 콘텐츠 업데이트가 필요할 때마다 서버에 전체 페이지를 다시 요청하는 대신
    JavaScript를 사용하여 필요한 부분만 동적으로 로드하거나 업데이트하는 방식

▶ SPA 의 장점
    빠른 페이지 로드
        초기 로드 시 전체 애플리케이션을 불러오므로 이후 페이지 전환 시 매우 빠른 응답 속도를 제공합니다.

    부드러운 사용자 경험
        페이지 전환 시 전체 페이지를 다시 로드하지 않으므로 사용자는 자연스럽고 연속적인 사용자 경험을 얻을 수 있습니다.

    효율적인 데이터 처리
        서버와의 통신은 필요한 데이터만 주고받으므로 네트워크 트래픽이 줄어들고 성능이 향상됩니다.

▶ vue-router
    vue.js의 공식 라우터 라이브러리, SPA를 손쉽게 만들 수 있도록 여러 기능을 제공한다.
